ProjectPool
=============
The g0v project pool.
[hackfoldr](http://hackfoldr.org/projectpool): 主要維護地
[Github Page](http://g0v.github.io/projectPool/): 備份

## Structure

- asserts
    + data
    + imgs
    + scripts
    + styles
- vendor
    + angular
    + jquery
- views

## Installation

`npm i`

`npm start`

and then open `http://localhost:3000/`
