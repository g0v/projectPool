ProjectPool
=============
The g0v project pool.

## Structure

- asserts
    + data
    + imgs
    + scripts
    + styles
- vendor
    + angular
    + d3js
    + jquery
    + semantic-ui
- views

## Installation

`npm i`

`npm start`

and then open `http://localhost:3000/`
